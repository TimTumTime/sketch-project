# Pages Documentation

This folder contains all of the web pages that the sketch-project has. This document is up-to-date as of 13/12/2025.

## Landing Page

The first page that users going onto the web application see. Serves as the entry point for the other features in the web application. Doesn't host any features beyond navigation.

### How to get to the page:

- Open the web application

### What pages it has access to (excluding in-browser features like previous page):

- Structured Learning
- Free Canvas

### What features does the page have:

- Navigation to other pages via buttons

## Free Canvas Page

A page avaliable to the user from the landing page. Serves as a practice ground for the user to do sketches without any restrictions or constraints. Hosts the standard features of the drawing components.

### How to get to the page:

- Open the web application
- From the landing page, click the button labeled "Free Canvas"

### What pages it has access to

- n.a (currently)

## Levels Page

A page describing all of the levels and proficiencies of sketching. Only hosts navigation

### How to get to the page:

- Open the web application
- From the landing page, click the button labeled "Structured Learning"

### What pages it has access to

- Topics Page

### What features does the page have:

- Navigation to other pages via buttons

## Topics Page

A page describing all of the topics within a level of sketching. Only hosts navigation

### What features does the page have:

- Navigation to other pages via buttons
- Use of a canvas component
